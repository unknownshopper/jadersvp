Sistema de Reservaciones y Control Operativo – Cafe Jade

Solución para registrar, organizar y operar reservaciones con control por roles, croquis de mesas y flujo de atención.

Objetivo

Implementar un sistema web para el control de reservaciones y operación diaria del restaurante, incluyendo selección de mesa mediante croquis, registro de clientes, y flujo de estados de atención, con control de acceso por roles.

Alcance funcional

  • Creación de reservaciones con fecha/hora (pasos de 15 minutos) y registro de datos del cliente.
  • Campo de personas (tamaño de grupo) para cada reservación.
  • Asignación opcional de mesa al crear la reservación.
  • Croquis (mapa) de mesas para seleccionar rápidamente mesa y filtrar por mesa.
  • Estados de reservación: WAITING, RESERVED, SEATED, COMPLETED, CANCELLED, NO_SHOW.
  • Estados de mesa: LIBRE, OCUPADA, RESERVADA, POR_LIMPIAR.
  • Vista de Hostess para alta de reservaciones y control de lista.
  • Vista de Caja para operación en tiempo real y cierre/liberación de mesas.
  • Vista Admin para métricas y supervisión operativa.
  • Configuración de WhatsApp API/WhatsApp Business para automatizaciones operativas (proveedor por definir).

Seguridad y roles

  • Inicio de sesión con Firebase Authentication (correo/contraseña).
  • Sesión segura con cookie httpOnly en el servidor.
  • Control de acceso por rol: ADMIN, DIRECTOR, HOSTESS, CAJA.
  • Permisos de operación:
    • HOSTESS: crear reservaciones y sentar/gestionar atención inicial.
    • CAJA: cierre de reservaciones y liberación de mesa.
    • ADMIN/DIRECTOR: acceso a administración y reportes.

Tecnología

Frontend
Next.js (App Router)
Backend
API Routes (Next.js) + Firebase Admin SDK
Base de datos
Cloud Firestore
Autenticación
Firebase Auth
Hosting
Firebase Hosting (configuración para producción)

Entregables

  • Sistema funcional desplegable para operación diaria.
  • Configuración de ambiente y variables para producción.
  • Configuración de WhatsApp API/WhatsApp Business para el flujo acordado (proveedor por definir).
  • Capacitación al personal tras la implementación.
  • Documentación operativa y técnica básica para uso y mantenimiento.
